# MoguMap 要件定義書

## 1. 目的

MoguMapは、ホットペッパーグルメAPIを利用して、利用者が飲食店を検索できるWebアプリです。

## 2. 最初の完成範囲

### 検索機能

- 対象は千葉県内の飲食店とする
- ホットペッパーグルメの中エリア単位で検索できる
- 中エリアの選択は必須とする
- 店名や料理名などのキーワードで検索できる
- キーワードの入力は任意とする
- 検索ボタンを押すと、条件に合う店舗を表示する
- 検索条件をURLに保持する
- 検索後にページを再表示しても、中エリアとキーワードをフォームへ反映する

### 検索結果一覧

- 1ページに10件表示する
- 各店舗について、次の情報を表示する
  - 店舗写真
  - 店舗名
  - 住所
  - 予算
  - 営業時間

### 店舗詳細

- 一覧から店舗を選ぶと、店舗詳細ページを表示する
- メイン写真を大きく表示する
- 店舗の基本情報を表示する

## 3. セキュリティ要件

- ホットペッパーグルメAPIのAPIキーをブラウザへ渡さない
- APIキーをGitへ保存しない
- APIキーをログや画面へ表示しない

## 4. 今後確認する項目

- 店舗詳細ページに表示する基本情報の具体的な項目
- 検索結果が0件だった場合の表示
- 外部APIでエラーが起きた場合の表示
- ページ番号をURLへどのように持たせるか
